SDK for Custom AI

Drop-in Python SDK with additional LangChain/LangGraph support. A few lines of code and every AI interaction flows into the detection pipeline. 

from stihia import StihiaClient, SenseGuard

client = StihiaClient()
guard = SenseGuard(client, messages)

async for chunk in guard.shield(llm.astream(prompt)):
    yield chunk  # secured by Stihia

Deterministic Classifiers

Known threat signatures: destructive tool usage, credential exposure, dangerous commands. Fires instantly with zero ambiguity.

Probabilistic Classifiers

ML models for semantic threats: prompt injections, social engineering, sophisticated data extraction that keyword matching cannot catch.

Anomaly Detection

Behavioral baselines catch scope creep, unexpected actions, and goal hijacking. Models adapt per project with no manual tuning.
